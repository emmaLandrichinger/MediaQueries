## Einführung
### Media Queries im CSS
```css
@media(Bedingung){
    /*Hier kommen die CSS Anweisungen hin,
      die interpretiert werden,
      wenn die Bedingung erfüllt ist*/
}
```
### Typische Bedingung
 
|Bedingung|Bedeutung|
|---|---|
|`max-width`|Stile gelten bis dieser Größe|
|`min-width`|Stile gelten ab dieser Größe|
