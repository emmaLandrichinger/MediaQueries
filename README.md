## Einführung
### Media Queries im CSS
```css
html, body{
    width: 100%;
    height: 100%;
    background-color: crimson;
}
 
@media (max-width: 1024px){
    html, body{
        width: 100%;
        height: 100%;
        background-color: darkolivegreen;
    }
}
```
### Typische Bedingung
 
|Bedingung|Bedeutung|
|---|---|
|`max-width`|Stile gelten bis dieser Größe|
|`min-width`|Stile gelten ab dieser Größe|
